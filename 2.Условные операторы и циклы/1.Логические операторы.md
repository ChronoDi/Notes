*Логические операторы*
======

За логические операции отвечает тип данных boolean
```cs
bool a = true;
bool b = false;

Console.WriteLine(a && b);  /** && - логическое И. Выражение будет истинным в том
                                случае, если оба выражения истина.
                                Вывод: False */

Console.WriteLine(a || b);  /** || - логическое ИЛИ. Выражение будет истинном в том
                                случает, если хотя бы одной из выражений истина.
                                Вывод: True */

Console.WriteLine(!a);      /** ! - логическое НЕ. Результатом будет обратное значение.
                                Вывод: False */
```

-----------

```cs
int a = 5;
int b = 7;

Console.WriteLine(a > b);   /** Операторы сравнения, могут быть >, <, >=, <=
                                Вывод: False */

Console.WriteLine(a == b);  /** == оператор равенства. НЕ ПУТАТЬ С = - ЭТО ПРИСВОЕНИЕ!
                                Вывод: False */

Console.WriteLine(a != b);  /** != оператор неравенства. Вывод: True */
```
