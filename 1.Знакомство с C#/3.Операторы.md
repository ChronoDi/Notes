*Операторы*
=======

Арифмитические операторы
```cs
+     // сложение.
-     // вычитание.
*     // умножение.
/     // делание.
%     // деление с остатком. Пример: 5 % 2 = 1. Ближайшее число которое делится на 2 без остатка - 4.
      // 5 - 4 = 1 - она и будет результатом от деления с остатком.

```
Логические операторы
```cs
==    // присвоение.
!=    // не равно.
>     // больше.
>=    // больше или равно.
<     // меньше.
<=    // меньше или равно.
```
Инкремент и декремент
```cs
int i;
i++;  // постфиксный инкремент.Сначала обрабатывается переменная, потом увеличивается на 1.
++i;  // префиксный инкремент. Сначала увеличивается на 1, потом обрабатывается.
i--;  // постфиксный декремент.Сначала обрабатывается переменная, потом уменьшается на 1.
--i;  // префиксный декремент. Сначала уменьшается на 1, потом обрабатывается.
```
